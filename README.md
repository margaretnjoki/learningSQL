# Learning SQL 🗄️
PostgreSQL practice, working toward Spring Boot.

## Rebuild the database
Run `schema/00_bookstore_setup.sql` against the `bookstore` database.

## Layout
- `schema/` — table definitions + seed data
- `dayN_*.sql` — queries per day
- `notes/` — lesson notes + cheat sheet

Dialect: PostgreSQL 15 · Tool: IntelliJ Ultimate Database console
